Create a Web Application that connects to a Windows Internal Root Certificate Authority, and all sub Certificate Authorities to create and manage list of all SSL certificates issued across a Windows Active Directory Domain. The application will track Web certificate expiration, and send expiration warnings to the Administrators of this application. This app should be able to create, or automate the creation of CSRs. This app, if possible, should also automate the placing and binding of certificates on windows servers. The infrastructure should consist of a MongoDB database, and Node.js/Express backend/frontend
